# PortfolioMySite

HTML / CSS / JavaScript の理解を深めながら制作している、ポートフォリオサイトの試作プロジェクトです。  
現在はトップページのタイルUIを中心に、レイアウト・色・質感・余白の調整を進めています。

## 目的

- semantic HTML の理解
- CSS Grid / Flexbox の理解
- UI / UX 設計の理解
- アニメーション実装の学習
- 自力で考えて実装する力を高めること

## 現在できていること

- CSS Grid を使ったタイル配置
- メインタイルとサブタイルの構成
- `clip-path` による角落とし
- OKLCH を使った色管理
- グラデーションと疑似要素による質感表現
- タイル素材の作成と配置検討

## 使用技術

- HTML
- CSS
- JavaScript
