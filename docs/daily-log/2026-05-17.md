## 学んだこと

- `querySelector()` は最初の1件だけを取得するため、同じ class が複数ある時は意図した要素か確認が必要。
- `z-index` は数値だけでなく、要素が重なり順へ参加できる状態かも重要で、`position` の有無が効く。
- `position: absolute` は、どの親要素を基準にしているかで表示位置が大きく変わる。
- スクロールバーの見た目を変える class と、実際に `overflow-y` を持つ class は別責務として分けられる。
- 共通デザインとページ固有レイアウトを分けると、同じ class を再利用しながら見た目を崩さずに済む。
