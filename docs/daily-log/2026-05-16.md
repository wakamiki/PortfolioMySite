## 学んだこと

- `querySelector()` は最初の1件、`querySelectorAll()` は一致する複数要素を取得する。
- `event` を使わないイベント処理では、アロー関数の引数は空でよい。
- `clip-path` を持つ要素の外側影は扱いづらく、影用ラッパーを分けると整理しやすい。
- Gridの直接子が変わると、`grid-column` / `grid-row` を持たせる要素も変わる。
- `opacity: 0` は見えなくなるだけでレイアウト上の場所は残る。
- `display: grid` を使う要素では、必要に応じて `grid-template-columns` も明示しないと、想定より狭い列になることがある。
- CSSは、重複整理、責務分離、状態class、変数管理を意識すると保守しやすくなる。
